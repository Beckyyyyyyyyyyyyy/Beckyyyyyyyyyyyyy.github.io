Code

Code is the tool and weapon for an individual to survive in the world. You can freely choose the way to achieve success. For example, a solid foundation in C language, the universal and pure nature of Python, the specialization of C#, and the rigorous power of C++.

Data Structures

To unlock a skill, you need to apply its characteristics. Use loop structures to simplify problems with patterns, use pointers to store memory and access variables to deal with problems based on experience, use stacks to manage various needs and desires, use recursion to achieve compound interest and high efficiency, and use control flow statements to design strategies.

Open Source

However, reading a hundred times is not as effective as going out and trying. You need to use the source code to implement projects and refine the fit through application.
